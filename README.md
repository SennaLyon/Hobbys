3.5 ABM模拟
本研究构建的 ABM 模型主要包括 代理主体（Agents）、外部环境（Environment）以及行为规则（Rules） 三个部分。模型以化工企业为核心决策主体，并引入政府政策和市场环境作为外部影响因素。
模型模拟时间步长设定为 1 年，模拟周期为10年，用于描述新型电催化技术从实验室研究到工业应用的长期扩散过程。在每一个时间步中，各代理主体根据经济收益和外部环境变化调整技术选择，从而形成技术扩散的动态过程。
3.1.1 代理主体设置
在模型中设置三类主要代理主体：
（1）工业企业代理（Industrial Firms）
工业企业是模型中的核心决策主体，代表潜在采用EGOR技术的化工生产企业。企业在生产过程中需要根据技术成本、能源消耗以及产品收益等因素决定是否采用尖晶石氧化物催化EGOR技术。
企业代理的主要状态变量包括：生产规模（Production capacity）、电流密度（Current density）、电解电压（Cell voltage）、催化剂寿命（Catalyst lifetime）、技术类型（Technology type）、企业利润（Profit）、风险感知（Risk perception）、技术采用概率（Adoption probability）
其中技术类型分为两种：传统技术（Traditional technology），即传统乙二醇氧化工艺；EGOR技术（EGOR technology），即采用尖晶石氧化物催化剂的乙二醇电氧化反应体系。
企业在每个时间步计算两种技术方案下的预期利润，并据此决定是否采用新技术。
（2）政府代理（Government）
政府代理用于模拟政策因素对技术扩散的影响。在能源转型和绿色化学发展过程中，政府往往通过补贴、碳税或环保法规等方式推动低碳技术的应用。
政府代理的主要参数包括：电价（Electricity price）、碳排放税（Carbon tax）、绿色技术补贴（Technology subsidy）、排放约束（Emission regulation）等。
这些政策变量会影响企业的生产成本，从而改变企业采用EGOR技术的经济吸引力。
（3）市场代理（Market）
市场代理用于描述相关化学品及能源价格变化对企业收益的影响。模型中主要考虑以下市场变量：乙二醇价格、乙醇酸市场价格、氢气价格、催化剂材料成本
由于EGOR反应能够同时生产高附加值化学品和氢气，因此市场价格变化对技术经济性具有重要影响。
